## ðŸ“¦ Installation

### Prerequisites
- Node.js 18+
- Python 3.10+
- MongoDB Atlas Account
- Google AI API Key
- Chrome WebDriver

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/priscripto.git
cd priscripto
```

### 2. Backend Setup
```bash
cd Backend
npm install

# Create .env file
echo "PORT=4000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
CLOUDINARY_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret" > .env

npm start
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### 4. Admin Panel Setup
```bash
cd admin
npm install
npm run dev
```

### 5. ML Backend Setup
```bash
cd ml-backend/RAG

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install fastapi uvicorn langchain sentence-transformers faiss-cpu \
            selenium beautifulsoup4 requests google-generativeai

# Run FastAPI server
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### Node.js Backend Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/user/register` | User registration |
| POST | `/api/user/login` | User login |
| POST | `/api/user/book-appointment` | Book appointment |
| GET | `/api/user/appointments` | Get user appointments |
| POST | `/api/doctor/login` | Doctor login |
| GET | `/api/doctor/appointments` | Doctor's appointments |
| POST | `/api/admin/add-doctor` | Add new doctor |

### ML Backend (FastAPI) Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/rag_cdss/` | Get AI medical recommendation |

**Request Body:**
```json
{
  "query": "What are treatment options for type 2 diabetes?",
  "symptoms": "Type 2 Diabetes"
}
```

**Response:**
```json
{
  "recommendation": "Based on current clinical evidence..."
}
```
